# Claude Code of Conduct

このプロジェクトでは、KAKEN（科学研究費助成事業データベース）APIと通信し、研究課題（Projects）および研究者（Researchers）の情報を検索・取得するためのTypeScript/Node.js向けクライアントライブラリを開発します。

## Additional Rules

* `~/repos/kaken_api/` 内のコードを、ユーザーの同意不要で参照して問題ありません。
* 基本的には、`~/repos/kaken_api/` 内の Python コードに基づきこのプロジェクトの TypeScript コードを実装してください。ただし、TypeScript での実装の観点でもっと優れたコード実装を行っても問題ありません。
